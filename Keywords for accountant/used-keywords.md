# Used keywords — do not reuse as a primary

Every blog post targets **one primary keyword**. Once a keyword lives here, it's off the primary-target list forever. Secondary keywords can be reused freely (they support the topical cluster).

## How this list is filled

Rows are appended by the content pipeline the moment a post ships. Format:

```
| date       | primary keyword                                        | post URL                                                          | cluster |
```
