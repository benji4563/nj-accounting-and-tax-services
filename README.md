# NJ's Accounting and Tax Services

Client project brain + website for **NJ's Accounting and Tax Services** — accounting and tax services for small businesses.

**Promise:** Focus on growth. We handle the finances.

---

## What's in this repo

| File / folder | What it is |
|---|---|
| `CLAUDE.md` | Project rules — read by Claude Code before any work |
| `project_specs.md` | Website blueprint (pages, tech stack, data models, done-criteria) |
| `brand-guidelines.md` | Full brand system (voice, colours, type, do's/don'ts) |
| `Skills/website-builder/SKILL.md` | Web/app build playbook — Next.js + Supabase + Tailwind + Vercel |

The website code itself will be added under `/app`, `/components`, `/lib`, etc. once `project_specs.md` is approved.

---

## Quick start (when the site exists)

```bash
npm install
npm run dev
# open http://localhost:3000
```

Make sure `.env.local` has the Supabase + Resend keys before running.
